# Vim-Config

### Install & Setup

Install _vim-plug_ from here: https://github.com/junegunn/vim-plug

Run the following:
```cd ~
rm -rf ~/.vimrc
rm -rf ~/.vim
git clone https://github.com/laurenabrahall/Vim-Config.git
vim ~./vimrc
```

Run the following inside vim:  
`:source %` and then `:PlugInstall`
